<h3 id="crossover-classes">Crossover Classes</h3><p>Any unlocked class from another <em>Gloomhaven</em> game can be chosen when creating a new character in <em>Frosthaven</em>.Conversely, any unlocked class from <em>Frosthaven</em> can be chosen when creating a new character in another <em>Gloomhaven</em> game. Classes may keep all of their previous enhancements, but doing so might cause imbalance. New character sheets for these crossover classes, which include traits and other useful additions, are available at <a href="https://cephalofair.com/frosthaven">cephalofair.com/frosthaven</a>.</p>
